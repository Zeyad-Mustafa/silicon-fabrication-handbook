# Presentations

This folder contains slide decks for talks and teaching sessions based on the handbook.

## Files

- `intro-to-cmos.pdf`: Full Intro to CMOS deck (10 slide-style sections).
  Topics include CMOS fundamentals, inverter behavior, FEOL/BEOL flow,
  scaling challenges, metrology, and design tradeoffs.
- `mems-overview.pdf`: Full MEMS overview deck (10 slide-style sections).
  Topics include MEMS fundamentals, surface/bulk micromachining,
  integration with CMOS, packaging, reliability, and calibration.
